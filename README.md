# API Search - AsyncAPI

This repository stores AsyncAPI event specification records for the [APIs.io](https://apis.io) search network.

## About

AsyncAPI records document event-driven and asynchronous API specifications indexed by the network. They are generated from api-evangelist repos and used to build the `_asyncapis/` Jekyll collection on the APIs.io network site.

## Related Repositories

- [apis](https://github.com/api-search/apis) - Individual API records
- [schemas](https://github.com/api-search/schemas) - JSON Schema references
- [providers](https://github.com/api-search/providers) - API provider records
- [network](https://github.com/api-search/network) - The APIs.io network site
