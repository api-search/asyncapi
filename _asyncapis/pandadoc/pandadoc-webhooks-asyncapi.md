description: The PandaDoc webhook system delivers real-time event notifications to registered subscriber endpoints when document lifecycle and platform events occur. PandaDoc sends HTTP POST requests containing JSON arrays of event objects to the target URL configured in a webhook subscription. Each event payload includes the trigger type and a data object describing the affected entity. Payloads are signed using HMAC-SHA256 with a per-subscription shared key, enabling receivers to verify authenticity before processing. Subscriptions can be configured to include optional payload extras such as fields, products, metadata, tokens, and pricing data alongside the base event information.
